    Filtrations #

    A decreasing filtration on a graded object A : ω → C is a family of subobjects F^s A^k ≤ A^k indexed by s : ℤ and k : ω, such that F^{s+1} A^k ≤ F^s A^k for all s, k.

    A decreasing filtration on a graded object A : ω → C. F s k is the subobject F^s A^k ≤ A^k. Monotonicity says F^{s+1} ≤ F^s (decreasing filtration).

    • F : ℤ → (k : ω) → CategoryTheory.Subobject (A k)

      The subobject F^s A^k of A k at filtration level s.

    • mono (s : ℤ) (k : ω) : self.F (s + 1) k ≤ self.F s k

      Filtration is decreasing: F^{s+1} ≤ F^s.

      A filtration is bounded if for each k, there exist bounds a ≤ b such that F^s A^k = A^k for s ≤ a and F^s A^k = 0 for s ≥ b.

      • lo : ω → ℤ

        Lower bound: for s ≤ lo k, the filtration is everything.

      • hi : ω → ℤ

        Upper bound: for s ≥ hi k, the filtration is trivial.

      • lo_le_hi (k : ω) : self.lo k ≤ self.hi k

        lo ≤ hi.

      • exhaustive (k : ω) (s : ℤ) : s ≤ self.lo k → fil.F s k = ⊤

        F^s = A for s ≤ lo.

      • hausdorff (k : ω) (s : ℤ) : self.hi k ≤ s → fil.F s k = ⊥

        F^s = 0 for s ≥ hi.

        Associated graded #

            Convergence #

            A spectral sequence E converges to a graded object A with filtration F if the E∞-page is isomorphic to the associated graded gr(A, F), up to a linear reindexing of the grading.

            This is weak convergence — the project does not consider strong convergence.

                    Detection #

                    An element y ∈ E∞^k detects an element x of A^{k'} of filtration ≥ s, meaning: x lies in F^s A^{k'}, and its image in gr^s A^{k'} = F^s/F^{s+1} corresponds to y under the convergence isomorphism.

                    Elements are modeled as generalized elements (morphisms from a test object T).

                    source

                    Prop

                    Detects conv y x means that the generalized element y : T ⟶ E∞^k detects the generalized element x : T ⟶ F^s A^{k'}, where (s, k') = conv.reindex k. Concretely: y ≫ iso(k) = x ≫ π where π : F^s → gr^s = F^s/F^{s+1}.
