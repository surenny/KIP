    SSData: nested subspace data at a single bidegree #

    source
    structure KIP.SpectralSequence.SSData (C : Type u) [CategoryTheory.Category.{v, u} C] [CategoryTheory.Abelian C] :
    Type (max u v)

    Nested subspace data for a spectral sequence at a single bidegree. Z r are the r-cycles (decreasing in r) and B r are the r-boundaries (increasing in r), both as subobjects of an ambient object V. Indexed by WithTop ℕ so that ⊤ represents the ∞-level.

    • V : C

      The ambient object (e.g., a graded piece of a chain complex)

    • Z : WithTop ℕ → CategoryTheory.Subobject self.V

      r-cycles as subobjects of V, decreasing: Z 0 ⊇ Z 1 ⊇ ... ⊇ Z ⊤

    • B : WithTop ℕ → CategoryTheory.Subobject self.V

      r-boundaries as subobjects of V, increasing: B 0 ⊆ B 1 ⊆ ... ⊆ B ⊤

    • Z_anti : Antitone self.Z

      Z is antitone: higher page index gives smaller cycle subobject

    • B_mono : Monotone self.B

      B is monotone: higher page index gives larger boundary subobject

    • Z_zero : self.Z 0 = ⊤

      Everything is a 0-cycle: Z 0 is the whole ambient object

    • B_le_Z (r : WithTop ℕ) : self.B r ≤ self.Z r

      Boundaries are always contained in cycles at each level

            Spectral sequences #

            source
            structure KIP.SpectralSequence.SpectralSequence (C : Type u) [CategoryTheory.Category.{v, u} C] [CategoryTheory.Abelian C] (ι : Type w) [AddCommGroup ι] [DecidableEq ι] :
            Type (max (max u v) w)

            A spectral sequence in an abelian category C with index type ι. Built on nested subspace data (SSData) at each bidegree.

            • ssData k provides the nested subspace witness at bidegree k.
            • Page objects E_r^k are computed as (ssData k).page ↑(r - r₀).toNat.
            • d r k is the differential d_r : E_r^k → E_r^{k + diffDeg r}.
            • d_comp_d expresses d² = 0 componentwise.
            • pageIso witnesses the isomorphism E_{r+1}^k ≅ E_{r+1}^k that comes from the homology identification. The full homology statement is given by SpectralSequence.pageHomologyIso below.
            • r₀ : ℤ

              Starting page index

            • ssData : ι → SSData C

              Nested subspace data witnessing the construction at each bidegree

            • diffDeg : ℤ → ι

              Degree of the differential d_r

            • d (r : ℤ) (k : ι) : (self.ssData k).page ↑(r - self.r₀).toNat ⟶ (self.ssData (k + self.diffDeg r)).page ↑(r - self.r₀).toNat

              Differential d_r^k : E_r^k → E_r^{k + diffDeg r}. The domain and codomain are computed from ssData.

            • d_comp_d (r : ℤ) (k : ι) : CategoryTheory.CategoryStruct.comp (self.d r k) (self.d r (k + self.diffDeg r)) = 0

              d_r ∘ d_r = 0 (componentwise)

            • pageIso (r : ℤ) (k : ι) : (self.ssData k).page ↑(r + 1 - self.r₀).toNat ≅ (self.ssData k).page ↑(r + 1 - self.r₀).toNat

              Isomorphism witnessing that consecutive pages are related. The mathematical content (homology of (E_r, d_r) ≅ E_{r+1}) is given by SpectralSequence.pageHomologyIso.

                    source
                    axiom KIP.SpectralSequence.SpectralSequence.pageHomologyIso {C : Type u} [CategoryTheory.Category.{v, u} C] [CategoryTheory.Abelian C] {ι : Type w} [AddCommGroup ι] [DecidableEq ι] (E : SpectralSequence C ι) (r : ℤ) (k : ι) :
                    E.Page (r + 1) k ≅ (E.pageShortComplex r (k - E.diffDeg r)).homology

                    BHS §0.1, Definition 0.3(5): Homology of (E_r, d_r) at index k is isomorphic to E_{r+1}^k.

                    Informal proof sketch: E_r = Z_r / B_r. The differential d_r has kernel Z_{r+1}/B_r and image B_{r+1}/B_r (by definition of the nesting). So H(E_r, d_r) = (Z_{r+1}/B_r) / (B_{r+1}/B_r) ≅ Z_{r+1}/B_{r+1} = E_{r+1} by the third isomorphism theorem.

                    The short complex is centered at source index k - diffDeg r, so the homology is at the middle term which is index k.

                    Axiomatized for general abelian categories. For AddCommGrpCat, this follows from QuotientAddGroup.quotientQuotientEquivQuotient.

                    The E∞-page of a spectral sequence, axiomatized as a field. For bounded/degenerate spectral sequences, this equals E_r for large r. For the general case, it is the intersection of permanent cycles modulo permanent boundaries.

                    • EInfty : ι → C

                      The E∞-page as a graded object

                    • toEInfty (r : ℤ) (k : ι) : self.ss.Page r k ⟶ self.EInfty k

                      Natural map from E_r to E∞ (for r sufficiently large in the degenerate case)

                    • toEInfty_compat (r : ℤ) (k : ι) : CategoryTheory.CategoryStruct.comp (self.ss.pageIso r k).hom (self.toEInfty (r + 1) k) = self.toEInfty (r + 1) k

                      The map to E∞ is compatible with the spectral sequence structure. Specifically, the maps from consecutive pages are compatible via pageIso.

                      axiom KIP.SpectralSequence.EInftyData.eInfty_isZero_of_page_isZero {C : Type u_1} [CategoryTheory.Category.{u_3, u_1} C] [CategoryTheory.Abelian C] {α : Type u_2} [AddCommGroup α] [DecidableEq α] (E : EInftyData C α) (k : α) (h : ∀ (r : ℤ), E.ss.r₀ ≤ r → CategoryTheory.Limits.IsZero (E.ss.Page r k)) :
                      CategoryTheory.Limits.IsZero (E.EInfty k)

                      BHS §0.1, Definition 0.4: If all pages E_r^k = 0 for r ≥ r₀, then E∞^k = 0. Informal: E∞ = Z∞/B∞. If E_r = Z_r/B_r = 0 for all r ≥ r₀, then Z_r = B_r. Since B_r ⊆ B∞ ⊆ Z∞ ⊆ Z_r, we get Z∞ = B∞, hence E∞ = 0.
